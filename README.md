# 📚 Online Bookstore Database (SQL Project)

## 📌 Project Overview
This project is a relational database system designed for an online bookstore. It manages data related to books, customers, orders, and order items, and performs data analysis using SQL queries.

---

## 🛠️ Tech Stack
- SQL (MySQL)
- Relational Database Design

---

## 📂 Database Structure
The project consists of the following tables:

- **Books** – Stores book details (title, author, category, price, stock)
- **Customers** – Stores customer information (name, email, city, etc.)
- **Orders** – Contains order details linked to customers
- **Order_Items** – Stores purchased book details for each order

---

## ⚙️ Key Features
- Designed normalized database schema with proper relationships
- Implemented primary and foreign keys for data integrity
- Inserted sample data for real-world simulation

---

## 📊 SQL Operations Performed
- Data Retrieval using SELECT statements
- Filtering using WHERE conditions
- JOIN operations to combine multiple tables
- Aggregation using SUM(), AVG(), COUNT()
- Grouping using GROUP BY and filtering using HAVING
- Sorting using ORDER BY
- Created views for simplified reporting

---

## 🔍 Insights Generated
- Identified top-selling books and categories
- Analyzed customer purchase behavior
- Calculated total sales and revenue
- Evaluated order trends and product performance

This project demonstrates strong understanding of SQL concepts including database design, data manipulation, and analytical query writing. It highlights the ability to extract meaningful insights from structured data.
